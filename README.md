# Учебный проект: консольный файловый менеджер на Python (`os` / `os.system`)

Простой учебный проект, демонстрирующий работу со стандартной библиотекой Python:

- `os` для файловых операций и определения платформы;
- `os.system` для запуска системных команд;
- `pathlib` для удобной работы с путями.

Главная цель проекта — **показать практическое использование библиотечных функций** без сложной логики и без лишней архитектурной нагрузки.

---

## Технологии и библиотеки

- **Python 3.10+** (проверено на Python 3.12);
- **Стандартная библиотека Python**:
  - `os`:
    - `os.system(...)`
    - `os.mkdir(...)`
    - `os.remove(...)`
    - `os.rmdir(...)`
    - `os.name`
  - `sys`:
    - `sys.platform`
  - `pathlib`:
    - `Path.cwd()`
    - `Path.iterdir()`
    - `Path.exists()`
    - `Path.is_dir()`
    - `Path.resolve()`

Проект не использует сторонние зависимости.

## Установка и запуск

### 1) Перейти в папку проекта

```powershell
cd "C:\Users\Windows\Desktop\os.sys"
```

### 2) Запустить скрипт

```powershell
python .\file_manager.py
```

Если в системе несколько версий Python, используйте:

```powershell
py .\file_manager.py
```

---

## Как пользоваться

После запуска:

1. Введите номер команды из меню;
2. Прочитайте результат;
3. Нажмите Enter для возврата в меню;
4. Для выхода выберите `0`.

## Кроссплатформенность

В проекте есть развилка по ОС:

- Windows:
  - `explorer`
  - `start powershell`
  - `cls`
- macOS:
  - `open`
- Linux:
  - `xdg-open`
  - `clear`

Это позволяет сохранить общий Python-код, меняя только конкретные команды ОС.
